# Task1
# Web Application Vulnerability Assessment & Reporting

## Project Overview
Performed a vulnerability assessment on *OWASP Juice Shop* to identify security flaws using ethical hacking tools and mapping findings to the *OWASP Top 10*.

## Tools Used
* *Scanner:* OWASP ZAP / Burp Suite
* *Target:* OWASP Juice Shop 
* *Documentation:* Professional Security Report

## Key Findings
* *Injection:* Identified SQL Injection vulnerabilities.
* *Broken Access Control:* Found unauthorized data exposure.
* *XSS:* Verified Cross-Site Scripting flaws.
